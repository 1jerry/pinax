 Developer Code Repo Workflow

Developers with Commit Privs
============================

Developers without Commit Privs
===============================

Login to http://github.com and search for Pinax or go directly here::

  http://github.com/pinax/pinax/tree/master

Use the 'fork' button to fork the Pinax tree to your area. This
invokes a "Hardcore Forking Action".

Grab the "Your Clone" URL from the page, using the Flash button if you
have Flash. It gives you something like this::

  git@github.com:cshenton/pinax.git

Clone this to your computer: in the current pinax-boot installation,
cd to src/ and then::

  git clone git@github.com:cshenton/pinax.git

Do your developement in src/pinax, test, later, rinse, repeat. When
you're satisfied with your changes, commit them to your local repo
copy and send it back up to your fork::

  git push

then issue a pull request to the committers
